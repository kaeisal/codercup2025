Linuxについて！もっと詳しく


まず「Linuxとは？」です。Linuxはリーナス・トーバルズ(Linus Torvalds)さんが大学院生だった時にリナックスの中身のカーネルを作りはじめオープンソースのOSとして誕生しました。 カーネルは本体なのでほかのシェル環境やパッケージ管理ソフトなどがセットになったのが上にある画像のubuntuやarchLinuxといったものです。リーナスさんが作ったのは中心核ということです。 Linuxにはいくつかの種類があって主にDebian系とRedhat系Arch系があります。(ほかにもありますが3つが一番有名です)

Debian系

最も派生ディストリビューションが多いLinux。その安定性や柔軟性から派生ディストリビューションの基礎や基盤となっています。デスクトップとしてもとてもおすすめで初心者でも使いやすい。
パッケージ管理システム: dpkg

派生ディストリビューション

  • Ubuntu

    • とてもユーザーが多くおすすめもされる。Linuxと言ったらこれ！Ubuntu派生のディストリビューションも多い。

  • Raspberry pi OS(旧Rasbian)

    • SBCのraspberrypi用の教育用OSです。標準ソフトも豊富でわかりやすいインターフェイス

  • Mint

    • Ubuntu派生のLinuxで人気のディストリビューション。

  • Kali

    • ハッキングやセキュリティー診断(侵入テスト)ソフトが豊富なハッカー愛用OS。法律に気を付けて使おう。

  • MX Linux

    • MX linuxは高速で低リソースで洗練されたデザインで軽量のOS。様々なハードウェアに対応しており安定している。

Redhat系

高いセキュリティー、安定性、企業向けのサポート体制が進んでいるので企業が安心して利用できます。Redhatは有償で安定性や信頼性が高く個人で扱う人はほとんどいません。
パッケージ管理システム: RPM

派生ディストリビューション

  • CentOS

    • Linuxの有償部分を取り除いたOSです。ほとんど差はありません。2021年をもって開発・サポートが終了しています。

  • Alama Linux / Rocky Linux

    • 高い互換性を持つ無料ディストリビューション

Arch系

シンプルさと柔軟性重視の軽量ディストリビューションです。ローリングリリースモデルを採用し常に最新のバージョンを保つことができます。自分で全部カスタイマイズ(設定)しないといけないので初心者にはあまりお勧めができません。
パッケージ管理システム: pacman

派生ディストリビューション

  • Manjaro

    • ArchLinuxの難しいインストールをGUIで簡単にしたディストリビューションです。初心者にも使いやすいです。

  • BlackArchLinux

    • KaliLinuxと同じくセキュリティー診断ソフトが豊富です。法律に気を付けて使いましょう。

  • SteamOS

    • PCゲームの販売などを行っているサービス。Valve Corporetionが開発したゲーム用OS。ゲーム重視のユーザー向け。
